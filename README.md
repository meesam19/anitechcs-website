# AnitechCS Website

A corporate website for AnitechCS, a global IT solutions provider.

## About

This is a professional website showcasing AnitechCS's services in website development, cloud hosting, and IT infrastructure. The company operates across multiple continents with offices in the US, Canada, Europe, Asia, and Africa.

## Features

- Responsive design that works on all devices
- Contact form for inquiries
- Newsletter subscription
- Service pricing information
- Interactive Google Maps
- Social media integration

## Tech Stack

- HTML5, CSS3, JavaScript
- Google Analytics for tracking
- Formspree for form handling
- GitHub Pages for hosting

## Structure

The site includes six main sections:
- Home with company overview
- About section with leadership info
- Services showcase
- Pricing tables
- Careers page
- Contact form with map

## Live Site

Visit: https://meesam19.github.io/anitechcs-website/

## Setup

This is a static website that runs on GitHub Pages. To use it:

1. Fork this repository
2. Enable GitHub Pages in Settings
3. Update the contact form email in index.html
4. Replace social media links with your own

## Contact Form

The contact form uses Formspree. After the first submission, you'll need to verify your email address. After that, all submissions will come through automatically.

## Security

- SSL certificate via GitHub Pages (Let's Encrypt)
- reCAPTCHA v3 for bot protection
- Client-side form validation

## Customization

To customize for your own use:
- Update company info in the About section
- Change pricing in the Pricing section
- Replace email addresses throughout
- Update social media links
- Add your own Google Analytics ID
